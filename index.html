Definiciones de CSS

1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de documentos HTML.

2. Funcionamiento Básico de CSS

CSS aplica reglas de estilo a los elementos HTML, especificando cómo deben presentarse en la pantalla.

3. Incluir CSS en un archivo HTML

CSS se puede incluir en HTML de tres maneras:

  • En línea: mediante el atributo style en una etiqueta HTML.
  • Interno: dentro de una etiqueta <style> en el <head>.
  • Externo: enlazando un archivo CSS con <link rel="stylesheet" href="estilos.css">.

4. Definición de una regla CSS

Una regla CSS define cómo se debe presentar un elemento HTML, compuesta por un selector y un bloque de declaración.

5. Elemento de una regla CSS

  • Regla: Conjunto de instrucciones que define un estilo.
  • Selector: Indica a qué elementos se aplica el estilo.
  • Bloque de declaración: Contiene pares de propiedad-valor, encerrados entre llaves {}.
  • Propiedad: Aspecto del estilo a modificar (por ejemplo, color).
  • Valor: Especificación de cómo se aplica la propiedad (por ejemplo, red).

6. Medios CSS

Herramientas que permiten aplicar estilos según el dispositivo, como @media para dispositivos móviles.

7. Comentarios CSS

Notas dentro del código CSS que no se ejecutan, escritas entre /* y */.

8. Sintaxis de la definición de las propiedades CSS

La sintaxis consiste en una propiedad seguida de un colon : y su valor, finalizando con un punto y coma ;. Ejemplo: color: blue;.
